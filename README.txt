How to fork a repository and submit git  commit  url by editing readme file

Step1:-Go to virtual lab repository and click n fork button on top right corner.

Step2:-On GitHub, navigate to your forked repository.

Step3:To clone the repository using HTTPS, under "Clone with HTTPS"

Step4:Open Git Bash.

Step5:git clone 'url'

Step6:Make  the changes

Step7:push the changes back to repository by:
 git add 'filename'
git commit -m "message"
git remote add origin "repo-url"
git push -u origin master
